FORMATEO DE TEXTO

Existen diferentes elementos para formatear un texto que se muestra en la página web, desde
la etiquetas p para indicar un párrafo, hasta otras propias de formateo como b para
indicar negrilla, em para emfatizar un texto, o del para borrar
texto, entre muchas otras listadas a continuación.

  1. b Texto en negrilla
  2. strong Texto importante
  3. i Texto en cursiva
  4. em Texto enfatizado
  5. mark Texto marcado
  6. small Texto opeque
  7. del Texto eliminado
  8. ins Texto insertado
  9. sub Texto subscripto
  10. sup Texto supercripto

HTML LINKS

Los links en HTML permite vincular enlaces a páginas dentro de la misma página web, como Bookmarks, o enlaces a otras páginas. Un link también puede ser configurado con una imagen o algún otro elemento de HTML

SINTAXIS

OTROS ATRIBUTOS

  • Target=" " indica como sera abierto el link, recibe 4 parámetros; _self: abre link en la misma página. _blank: abre link en una nueva ventana. _parent: abre link en un paren frame. _top: abre link en todo el body de la ventana.
  • Absolute URLs: referencia un url de una página externa, copiando todo el link "https://www..."
  • Relative URLs: referencia a un archivo o página alojada internamente en las carpetas del proyecto.
  • Imagen como link: usando el elemento img dentro de a href

HTML IMAGENES

Las imagenes permiten mejorar visualmente el aspecto de una página web, ya sea usando Imagenes como background, elementos que redireccionen a otras páginas, etc.

SINTAXIS

  • El tag <img>, es usado para incrustar una imagen.
  • El atributo src especifica el link de la imagen.
  • El atributo alt especifica un texto alterno a la imagen.
  • Los atributos width y height permiten modificar el alto y ancho de una imagen.

IMAGES MAP

Con image maps es posible crear zonas de una imagen que pueda ser clickeables. Para ellos se debe usar el atributo usemap y el valor anteponiendo un #

...usemap="#workmap">
Para demarcar las áreas de la imagen, se usa la etiqueta <area> dentro de <map name="workmap">

Shapes

El atributo <area shape=" " recibe 4 valores:

  • rect: define una área rectangular
  • circle: define una región circular
  • poly: define una región poligonal
  • default: define una región entera

Coords

Las coordenadas en caso de los atributos rect y poly se realiza en pares, uno para el eje X y otro para el eje Y. Para el atributo circle los dos primeros valores indican el centro del círculo, y el tercer valor el radio.

BACKGROUND IMAGES

Un background puede ser especificado casi en cualquier elemento HTML por medio del atributo style de CSS background-image

Por medio de CSS se pueden aplicar diferentes propiedades para modificar la imagen.

HTML LISTAS

Con las listas de HTML se puede relaciones información en un grupo de ítems o listas, que pueden ser ordenadas de forma numerica, alfabetica o sin orden.

LISTAS ORDENADAS

Inician con el tag <ol>, y el tag <li> para los ítems
El tipo de marcador se puede modificar con los siguientes valores:
  1. type="1": lista numerica, valor por defecto
  2. type="A": Lista se representa con letras mayúsculas
  3. type="a": Lista se representa con letras minúsculas
  4. type="I": Lista se representa con romanos en mayúsculas
  5. type="i": Lista se representa con romanos en minúsculas

LISTAS DESORDENADAS

Inician con el tag <ul>, y el tag <li> para los ítems
El marcador puede ser modificado aplicando la propiedad de CSS list-style-type aceptando los siguientes valores:
  • disc: valor por defecto
  • circle: el marcador es un circulo
  • square: el marcador es un cuadrado
  • none: la lista no tendra marcador

LISTAS DESCRIPTIVAS

Inician con el tag <dl>, el tag <dt> determina un nombre a la lista, y <dd> para los ítems
